* Ambient Music

## Usage

Send messages to [@pi20182_atp_bot](https://t.me/pi20182_atp_bot) with your telegram account.

### Dice Roller

#### Roll

Command template:
```
roll 2d20
```
This returns the result from the roll of 2 dices of 20 faces. If you want the individual result of each dice:

```
roll 2d20 all
```
